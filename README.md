### Biomedical Engineering Undergraduate at the University of Moratuwa

As a passionate Biomedical Engineering undergraduate based in Sri Lanka, my focus lies at the intersection of healthcare, IoT, and data-driven solutions. My project experience ranges from developing smart embedded systems like an IoT inhaler monitor to creating practical analog circuits for audio systems. I enjoy applying my expertise in signal processing, machine learning, and hardware design (such as SolidWorks and PCB design) to build impactful, real-world applications. Whether it's analyzing health data for actionable insights or optimizing physical enclosures for manufacturability, I am committed to exploring innovative technologies that improve healthcare outcomes.

- Signal Processing & Data Analysis
- Embedded Systems & Hardware Design
- Machine Learning Applications
- 3D Modeling and Product Design

## 🚀 Featured Projects

### [Smart Breath Buddy](https://github.com/Pamodya-Wijesinghe/smart-breath-buddy) *(Sep 2025 – Nov 2025)*
An IoT-enabled healthcare project designed to monitor inhaler usage and inhalation behavior.
- Developed ESP32 firmware for sensor data acquisition and secure cloud transfer.
- Designed the web application for device management and data visualization.
- Integrated the complete device → cloud → web data pipeline.

### [Diabetic Patient Data Analysis](#) *(Oct 2025 – Present)*
Analyzed diabetic patient datasets to identify patterns related to disease progression.
- Performed data cleaning, exploratory data analysis, and visualization to extract insights supporting data-driven healthcare decisions using Python.

### [UroSense](https://github.com/Pamodya-Wijesinghe/urosense-app) *(Mar 2025 – May 2025)*
An ICU device that continuously monitors urine output (volume via load cells) and detects abnormal color (via RGB sensor), alerting staff through a mobile app and Firebase backend.

### [Analog Electronics Project | Audio Voice-Over Device](https://github.com/Pamodya-Wijesinghe/Audio-voice-over-analog-project) *(Sep 2025 – Nov 2025)*
Designed an audio voice-over circuit that prioritizes microphone input over a primary audio source.
- Led the enclosure design using SolidWorks, focusing on manufacturability, component placement, and optimizing structure to reduce material waste.
